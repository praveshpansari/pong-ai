# pong-ai

## Installation
1. Download all the files and Love2D
2. run cmd `love Pong`

## Features

1. AI Player 2
2. Retro look
3. Difficulty scales with time (ball velocity gets faster)
4. 2 Players with (WASD and arrow keys)
